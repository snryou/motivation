expert_info.csv: The file contains the initial information of 120 experts. Each line in the file represents accumulated information for an expert in a data set. 
Id: expert user ID.
CreationDate: expert user registration date.
LastAccessDate: expert user's last access date.
DownVotes: total downvotes of an expert user.
UpVotes: total upvotes of an expert user.
Reputation: the total reputation of an expert user.
Views: total views of an expert user.
WebsiteUrl: Links to personal pages of expert users.

motivation_variable_info.csv: The file contains all the information on the motivation variables of 120 experts. Each line in the file represents motivation information for an expert in a data set. 
Id： expert user ID.
status: status level.
reputation_s: reputation points corresponding to status levels
dt_0: expert user registration date.
dt_s: the date of acquisition of status level from estimation.
post: the average daily number of posts created at certain status.
badge: the average daily number of badges obtained at certain status.
reputation: the average daily number of reputation points obtained at certain status.
learning: the average daily number of views at certain status.
reciprocity: the average daily number of votes at certain status.
url: whether the expert has a personal page or not?
day: days since registration.
